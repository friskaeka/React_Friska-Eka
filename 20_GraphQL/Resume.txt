A. GraphQL
   GraphQL adalah suatu quary language untuk API dimana kita bisa mendefine data yang ingin kita kirim ke klien dan kita bisa meminimal data. 
ada 3 fitur di graphQL :
1. Query : cara untuk mendapat data
2. Mutation : manipulasi data
3. Subscription : mendapatkan data paling update
